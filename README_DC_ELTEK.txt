DC Eltek — изменения

Что добавлено:
1) gui/pages/dc_eltek_page.py
   Новая вкладка DC Eltek:
   - выбор папки проекта;
   - автозаполнение клиента из пути после 02_Projects;
   - ручное редактирование клиента;
   - выбор Excel calc;
   - загрузка списка листов из .xlsx/.xlsm;
   - выбор шаблона КП .docx;
   - предпросмотр заполненных полей;
   - кнопка "Сформировать КП" пока показывает сообщение, генерацию подключим следующим этапом.

2) brands/dc_eltek.py
   Обновлена заглушка бренда и добавлена функция извлечения клиента из пути.

3) apply_dc_eltek_patch.py
   Скрипт для аккуратного подключения вкладки DC Eltek в gui/main_window.py.

Как применить:
1) Скопируй папки gui и brands из архива в корень проекта с заменой файлов.
2) Скопируй apply_dc_eltek_patch.py в корень проекта.
3) Запусти из корня проекта:
   python apply_dc_eltek_patch.py

После этого в программе должна появиться вкладка DC Eltek.
