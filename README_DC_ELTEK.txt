DC Eltek — полный пакет изменений с генерацией Word-КП
======================================================

Этот архив уже содержит готовые файлы для замены. Патч-скрипт запускать не нужно.

Состав архива:
- gui/main_window.py
- gui/pages/dc_eltek_page.py
- brands/dc_eltek.py
- brands/registry.py
- templates/dc_eltek/Offer_Company_02-07-26_TAGGED_Eltek.docx

Как применить:
1. Распаковать архив в корень проекта offer-generator.
2. Разрешить замену файлов.
3. Запустить приложение.

Что добавлено:
- вкладка DC Eltek подключена напрямую в gui/main_window.py;
- выбор папки проекта;
- клиент автоматически берётся из пути после 02_Projects;
- поле клиента можно редактировать вручную;
- выбор Excel calc;
- выбор листа для КП;
- выбор Word-шаблона КП;
- автоматический поиск шаблона в templates/dc_eltek и templates;
- универсальный парсер Excel без жесткой привязки к строкам и колонкам;
- строки модели, количества, цены, НДС и итоговой суммы ищутся по заголовкам/содержимому;
- предпросмотр позиций;
- нижний блок итогов: количество позиций, общее количество, сумма без НДС, НДС, общая сумма с НДС;
- кнопка «Сформировать КП» теперь создает Word-файл по шаблону;
- Word-теги заполняются автоматически:
  {{offer_date}}, {{offer_version}}, {{client_company_full}}, {{intro_text}},
  {{unit_price_header}}, {{total_price_header}}, {{item_no}}, {{item_name}},
  {{item_qty}}, {{item_unit_price}}, {{item_total}}, {{total_label}}, {{grand_total}},
  {{total_price_block}}, {{payment_terms}}, {{delivery_time}}, {{delivery_terms}},
  {{installation_terms}}, {{startup_terms}}, {{offer_validity}}, {{currency_terms}},
  {{signer_name}}, {{signer_position}}, {{manager_name}}, {{manager_position}},
  {{manager_email}}, {{manager_phone}}.

Куда сохраняется КП:
- в выбранную папку проекта DC Eltek;
- имя файла вида offer_КЛИЕНТ_дд-мм-гг_revN.docx;
- revN автоматически увеличивается, если в папке уже есть КП с rev.
