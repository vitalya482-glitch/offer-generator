# Auto output folder patch

Назначение патча: при выборе новой папки проекта программа автоматически подставляет папку результата.

## Что заменить

Скопируйте файлы из этого архива в корень проекта с заменой:

- gui/main_window.py
- gui/path_helpers.py

## Как работает

1. Пользователь выбирает другую папку проекта.
2. Старый путь в поле "Папка результата" очищается.
3. При сканировании проекта программа ищет папку вида:
   - 02_Sales docs
   - 03_sales docs
   - Sales docs
   - любой другой вариант, где в имени есть слова sales и docs.
4. Если папка найдена, она автоматически подставляется в поле "Папка результата".
5. Если папка не найдена, используется сама папка проекта.

## Важно

Ручной выбор папки результата по-прежнему работает. Он сохраняется до следующей смены папки проекта.

## Проверка

Откройте программу, выберите другую папку проекта. Поле "Папка результата" должно автоматически смениться на папку Sales docs, например `03_sales docs`.
